# ts-frontend

## install

```sh
git clone <repository> <dir>
cd <dir>
yarn
```

## develop

```sh
yarn dev
```

## type check

```sh
yarn type-check
```

## lint

```sh
yarn lint
```

## test

Only node-like environment is available.

```sh
yarn test
```

## build

```sh
yarn build
```

## deploy

```sh
yarn deploy
```
